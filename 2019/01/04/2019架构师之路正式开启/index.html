#开源框架解析篇

To Be One ~ Spring

什么是Spring框架？Spring框架有哪些主要模块？

答：
1、为了解决企业应用开发的复杂性而创建的.
2、主要包括7中模块：spring core、spring aop、spring context、spring web、spring web mvc、spring orm、spring dao.

Spring框架带来哪些好处？

答：
1、最大优点就是开发中无需关注原始技术上的问题，只需关注核心业务。
2、Spring能降低不同模块之间的耦合度，不同模块之间信息的沟通是通过对象传递完成的，而对象能否顺利传递就是要合理的构建好对象，而管理好对象的构建方式就能管理好对象传递。

什么是控制反转（IOC）？什么是依赖注入？

答：
1、最大优点就是开发中无需关注原始技术上的问题，只需关注核心业务。
